# Student Record Management System in C
This is a simple C program that stores student details using structures.
It calculates and displays the percentage of each student.
## Features
- Uses structure and structure array
- Takes multiple student inputs
- Calculates average/percentage
- Displays formatted output
## Concepts Used
- Structures
- Arrays
- Loops
- scanf / printf
- Float formatting
## Sample Output
Roll No: 1290  
Name: Meenakshi  
Percentage: 72.67
